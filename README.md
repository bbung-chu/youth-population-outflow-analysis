# 청년은 왜 특정 지역을 떠나는가?

> 전국 시군구별 청년 인구 이동 데이터를 기반으로 청년 인구 유출 현상을 분석하고, 청년들이 특정 지역을 떠나는 이유와 지역별 유출 특성을 파악하는 데이터 분석 프로젝트

---

## 📌 Project Overview

최근 청년 인구의 수도권 및 특정 지역으로의 집중은 지역 간 인구 불균형과 지방소멸 문제로 이어지고 있다.

본 프로젝트에서는 전국 시군구별 청년 인구 이동 데이터를 분석하여 청년 인구가 많이 유출되는 지역을 식별하고, 이후 청년들의 이동 경로와 이동 사유 및 지역적 특성을 분석하여 청년 인구 유출의 원인을 데이터 기반으로 파악하고자 한다.

### 핵심 질문

> **청년들은 어떤 지역을 떠나며, 어디로 이동하고, 왜 떠나는가?**

---

## 🎯 Project Objectives

- 전국 시군구별 청년 인구 순이동 현황 파악
- 청년 인구 순유출이 지속되는 지역 식별
- 청년 인구 유출 지역의 이동 경로 분석
- 청년 인구의 주요 이동 사유 분석
- 청년 유출 지역의 지역적 특성 분석
- 청년 인구 유출과 지역 특성 간 관계 분석
- 지역별 청년 인구 유출 원인 도출
- 분석 결과를 기반으로 청년 인구 유출 대응 방안 제안
